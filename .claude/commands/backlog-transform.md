# Command: Backlog Transform

Execution OKR Table을 입력하면
Frontend / Backend / Data Analytics Backlog로 변환한다.

---

# Transformation Flow

1. Execution OKR의 핵심 행동을 추출한다
2. 행동을 Frontend / Backend / Data Analytics로 책임 분리한다
3. K, R 지표를 이벤트 수집/계산/분석 행동으로 변환한다
4. Actor 기반 Backlog Title 생성

---

# Actor 목록

**Frontend**

- 애플리케이션
- 추천 페이지
- 콘텐츠 페이지

**Backend**

- 백엔드 서버
- API 서버
- 시스템

**Data Analytics**

- Analytics 시스템
- Metric Engine
- Experiment Engine

---

# Backlog 유형

**Frontend**

- Behavior Backlog
- UI Backlog

**Backend**

- Behavior Backlog

**Data Analytics**

- Behavior Backlog

---

# 최종 출력

- Frontend Backlog Titles
- Backend Backlog Titles
- Data Analytics Backlog Titles
